Reference Manager Sync

BibTeX

Click BibTeX on any paper card to copy a single entry. The toolbar BibTeX button exports all visible papers at once.

Mendeley

Connect your Mendeley account via OAuth in Settings → Automation. Once connected, use Sync Saved Papers to push your library.

Zotero

Enter your Zotero API key and user ID in Settings → Automation. You can sync to a specific Zotero collection if you like.

Smart Features

Hybrid Search

Use the search bar to find papers by keyword, semantic meaning, or both. Toggle between Keyword, Semantic, and Hybrid modes using the dropdown next to the search box. Hybrid mode combines BM25 full-text search with SPECTER2 embedding similarity for the best results.

Ranking Explanations

Each paper shows why it was recommended. Expand the score details to see reasons like “Matched author: Kaiming He”, “Highly cited (342 citations)”, or “Similar to saved: NeRF in the Wild”. Look for the amber bullet points below the score breakdown.

Citation Counts

During each scrape, citation data is fetched from Semantic Scholar. Papers show a blue citation badge you can click to see the full page. Sort by Most Cited.

OpenAlex Enrichment

Papers are enriched with data from OpenAlex: topic classifications, open-access status, and additional citation counts. Enable or configure in config.yaml under openalex.

Priority & Shared

Beyond Save and Skip, you can mark papers as Priority for must-reads or Shared when you send them to colleagues. These stronger signals improve future recommendations more than a regular save.

Follow Author / Mute Topic

Expand a paper’s details to find Follow and Mute buttons. Follow adds the author to tracked authors. Mute suppresses papers with that topic tag.

Related Papers

Expand a paper’s details to see links to similar papers from your library. Similarity uses SPECTER2 scientific paper embeddings for high-quality semantic matching, with TF-IDF fallback when embeddings are unavailable.

Duplicate Detection

If a new paper has a nearly identical title to one already in the database, it gets an orange Dup badge so you know at a glance.

Historical Search

Use the Discover page in the top nav to search arXiv by date range and category. Matched papers are added to your inbox.

Scheduled Scrapes

Enable automatic daily scrapes by adding a scheduler section to your config.yaml. The app runs scrapes at your chosen time without needing cron.

Saved Searches

Bookmark your filter combinations as saved searches. Set categories, include/exclude keywords, author filters, date windows, and citation thresholds. Saved searches appear as quick-access pills on the dashboard and can be re-run at any time.

Corpus Analytics

Explore your paper library at a higher level. Topic clusters group recent papers by theme. Emerging topics compare recent trends against a baseline window to detect new directions. Neighbor papers find undiscovered papers similar to a collection or seed set.

BibTeX Export

Click the BibTeX button on any paper card to copy a single citation entry, or use the toolbar button to export all visible papers at once. Bulk export is also available via GET /api/export/bibtex.

HTML Report Export

Export your current paper view as a standalone HTML report for sharing or offline reading. Available via the API at GET /api/export or the CLI with python export_cli.py.

PDF Thumbnails

Paper cards can display first-page PDF thumbnails for visual scanning. Thumbnails are generated automatically during scrapes and can be backfilled for existing papers with cv-arxiv-backfill thumbnails.

Multiple Feed Sources

Monitor additional arXiv categories or custom RSS feeds beyond cs.CV. Add, remove, and toggle feed sources from the Settings page or via the /api/feed-sources API.

Advanced Filters

Click Advanced Filters below the search bar to filter by arXiv category, resource availability, reading status, or to show hidden papers.

Saved Searches

After setting up filters, click Save search to bookmark the combination. Saved searches appear as quick-access pills.

Bulk Operations

Click Select in the toolbar to enter bulk mode. Select multiple papers and save or skip them all at once.
