Common Questions

Do I need Gmail?

No. Gmail is only for sending yourself digests by email. You can use the app normally without setting it up.

Do I need the AI features?

No. They are optional. The app can still match and rank papers using your authors, labs, and topics.

Why is my inbox empty?

Usually this means your tracked terms are very narrow, or there were no matches in the selected timeframe. Try broader topics or switch to a wider timeframe.

What should I learn first if I am not technical?

Learn just three things: add interests, run a scrape, and review papers with Save or Not Interested. Everything else can wait.

What is SPECTER2?

SPECTER2 is a scientific document embedding model from the Allen Institute for AI. It converts paper titles and abstracts into numerical vectors that capture semantic meaning, enabling high-quality similarity comparisons and semantic search. The model is downloaded automatically on first use (~420MB).

How much disk space do embeddings use?

The FAISS vector index uses roughly 3KB per paper. For a typical library of 5,000 papers, the index is about 15MB. The SPECTER2 model itself is a one-time ~420MB download stored in ~/.cache/huggingface/.

What is OpenAlex?

OpenAlex is an open catalog of scholarly works. This app uses it to enrich papers with topic classifications, open-access status, and additional citation counts. It’s enabled by default and free to use (no API key needed).

What’s the difference between keyword and semantic search?

Keyword search (BM25) finds papers containing exact words you type. Semantic search uses SPECTER2 embeddings to find papers by meaning — so “image classification” also finds papers about “visual recognition”. Hybrid mode combines both approaches using Reciprocal Rank Fusion for the best of both worlds.
