Settings Overview

You can leave most settings alone at first. The only section you need for a useful first run is Research Setup.

Start Here

Research Setup

Add the authors, labs, and topic keywords you care about. One item per line. This is what decides what lands in your inbox.

Optional

Ranking

Fine-tune how strongly the app reacts to authors, labs, topics, freshness, citations, AI relevance, and muted terms. Most users can keep the defaults until they have some feedback history.

Optional

Automation

Email digests, scheduled scrapes, Mendeley/Zotero sync, feed sources, and monitoring for recent automation runs.

Optional

LLM / AI

AI-powered summaries and relevance scores. Useful if you want richer summaries or another ranking signal, but not required. See CLI → AI Summaries for setup details.

OpenAlex

Enabled by default. Papers get topic classifications, open-access status, and citation counts from OpenAlex, which makes filtering and browsing more useful without extra setup.

openalex:
  enabled: true        # set to false to disable
  email: ""            # your email for the polite pool (higher rate limits)
