UI Guide & Main Buttons

Run Scrape

Checks arXiv for recent papers and refreshes your inbox.

Inbox

Shows papers that matched your interests and still need review.

Saved

Your keep list. Use this for papers worth coming back to.

Save

Marks a paper as important and keeps it in your shortlist.

Upvote

Tells the app, "this is relevant." It helps similar papers move up.

Not Interested

Hides a weak match and teaches the app to show less of that kind of paper.

Paper Organization

Ways to keep your reading list tidy beyond Save and Not Interested.

Collections

Group papers into named folders (e.g. “Thesis Ch. 3”). Create collections, then use the folder icon on any paper card to file it. Filter the dashboard by collection using the pills at the top.

User Tags

Add custom tags to any paper with the + tag button. Tags appear as indigo chips and can be removed by clicking the × next to them.

Notes

Each paper has a free-text notes field (expand More details to see it). Notes auto-save as you type.

Reading Status

Mark papers as To Read, Reading, or Read using the dropdown on each card. You can filter by reading status in Advanced Filters.
