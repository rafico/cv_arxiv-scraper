Search

Use the search bar at the top of the dashboard to find papers. Toggle between three modes using the dropdown next to the search box.

Keyword

Finds papers containing the exact words you type. Uses BM25 full-text search under the hood.

Semantic

Finds papers by meaning. “image classification” also matches “visual recognition”. Powered by SPECTER2 embeddings.

Hybrid

Combines keyword and semantic results for the best of both. This is the default mode.

Filters & Sorting

Match type filter

Show only Author, Affiliation, or Title matches using the pills below the search bar.

Advanced Filters

Click Advanced Filters to filter by arXiv category, resource availability, reading status, or to show hidden papers.

Sort options

Sort by score (trending), date (newest), recommendations, or citation count.

Time windows

Switch between daily, weekly, monthly, or all-time views.

Saved Searches

Bookmark a filter combination so you can re-run it with one click.

Creating

Set up your filters, then click Save search. Give it a name and optional criteria like categories, date windows, or citation thresholds.

Using

Saved searches appear as quick-access pills on the dashboard. Click one to instantly apply its filters.

Discover (Historical Search)

Use the Discover page in the top nav to search arXiv by date range and category. This lets you backfill papers from the past — not just today’s feed. Matched papers are added to your inbox like any other scrape.
