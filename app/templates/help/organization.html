Collections

Group papers into named folders such as “Thesis Ch. 3”, “Survey candidates”, or “Lab Meeting”. A paper can belong to more than one collection.

Create a collection

Use the collection manager to create named, color-coded folders for projects, reading lists, or meetings.

Add papers

Click the folder icon on any paper card to file it into one or more collections. Then filter the dashboard by collection when you want to work on one project at a time.

Use collections when you want a paper to live inside a specific project or bucket.

Tags

Add custom tags with the + tag button. Tags are best for cross-cutting labels like “methods”, “datasets”, or “to-cite” that may apply across many collections.

Use tags when you want lightweight labels that cut across projects, not separate folders.

Notes

Each paper has a free-text notes field under More details. Use it for takeaways, quotes, open questions, or reminders. Notes auto-save as you type.

Reading Status

Track where each paper is in your reading pipeline.

To Read Reading Read

Set the status from the dropdown on each paper card, then filter by reading status in Advanced Filters when you want a reading queue.

Follow & Mute

Follow an author

Expand a paper’s details and click Follow. The first author is added to your tracked authors list so future papers from them rank higher.

Mute a topic

Click Mute when a topic keeps producing noise. Future papers with that topic tag are pushed down or suppressed.
