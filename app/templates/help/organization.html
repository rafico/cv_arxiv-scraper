Collections

Group papers into named folders (e.g. “Thesis Ch. 3” or “Lab Meeting”). A paper can belong to multiple collections.

Create a collection

Use the collection manager to create named, color-coded folders.

Add papers

Click the folder icon on any paper card to file it into one or more collections. Filter the dashboard by collection using the pills at the top.

Tags

Add custom tags to any paper with the + tag button. Tags appear as indigo chips and can be removed by clicking the × next to them. Use tags for cross-cutting categories like “methods”, “datasets”, or “to-cite”.

Notes

Each paper has a free-text notes field. Expand More details on a paper card to see it. Notes auto-save as you type.

Reading Status

Track your reading progress on each paper.

To Read Reading Read

Set the status using the dropdown on each card. Filter by reading status in Advanced Filters.

Follow & Mute

Follow an author

Expand a paper’s details and click Follow. The first author is added to your tracked authors list.

Mute a topic

Click Mute to suppress papers with that topic tag in future rankings.
