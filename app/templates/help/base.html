Use these pages to get from a first scrape to a repeatable paper-review workflow. If you are new, start with Quick Start, then Settings, then the UI Guide.

New here? Most users only need three things to begin: add interests, run a scrape, and review papers with Save or Not Interested. Gmail, reference-manager sync, CLI tools, and AI features can wait until later.
