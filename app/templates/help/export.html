BibTeX Export

Use BibTeX export when you want citation entries immediately, without setting up a reference-manager sync.

Single paper

Click BibTeX on any paper card to copy the citation entry to your clipboard.

Bulk export

Use the toolbar BibTeX button to download all visible papers as a .bib file. Also available in bulk-select mode.

Reference Manager Sync

Use sync when you want papers you save in the app to move into your existing reference library.

Mendeley

Connect your Mendeley account via OAuth in Settings → Automation. Once connected, use Sync Saved Papers to push your saved papers into your library.

Zotero

Enter your Zotero API key and user ID in Settings → Automation. You can choose a specific Zotero collection as the destination.

HTML Report

Export your current paper view as a standalone HTML file when you want to share a snapshot or read it offline.

  • Via the API: GET /api/export?timeframe=daily
  • Via CLI: python export_cli.py --timeframe weekly

Daily Email Digest

Get matched papers delivered to your inbox automatically. This is optional and only matters if you want the app to email you instead of opening the dashboard.

Setup

1

Create Gmail credentials

In Google Cloud Console, create a project, enable the Gmail API, and create an OAuth 2.0 Client ID. Download the credentials JSON.

2

Upload credentials

Upload the JSON in Settings → Automation, or save it as credentials.json in the project root.

3

Authorize

Complete the OAuth flow in Settings, or run python gmail_auth_setup.py if you are setting this up on a headless machine.

4

Set recipient & test

Set email.recipient in config.yaml, then run cv-arxiv-digest --dry-run to preview the digest before sending anything.

Security

Only the gmail.send scope is requested — the app cannot read, list, or delete your emails. Tokens are stored locally with restricted permissions.
