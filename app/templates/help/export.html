BibTeX Export

Single paper

Click BibTeX on any paper card to copy the citation entry to your clipboard.

Bulk export

Use the toolbar BibTeX button to download all visible papers as a .bib file. Also available in bulk-select mode.

Reference Manager Sync

Mendeley

Connect your Mendeley account via OAuth in Settings → Automation. Once connected, use Sync Saved Papers to push your library.

Zotero

Enter your Zotero API key and user ID in Settings → Automation. You can sync to a specific Zotero collection.

HTML Report

Export your current paper view as a standalone HTML file for sharing or offline reading.

  • Via the API: GET /api/export?timeframe=daily
  • Via CLI: python export_cli.py --timeframe weekly

Daily Email Digest

Get matched papers delivered to your inbox every morning — no need to open the dashboard.

Setup

1

Create Gmail credentials

In Google Cloud Console, create a project, enable the Gmail API, and create an OAuth 2.0 Client ID. Download the JSON file.

2

Upload credentials

Upload the JSON in Settings → Automation, or save it as credentials.json in the project root.

3

Authorize

Complete the OAuth flow in Settings, or run python gmail_auth_setup.py for headless environments.

4

Set recipient & test

Set email.recipient in config.yaml, then run cv-arxiv-digest --dry-run to preview.

Security

Only the gmail.send scope is requested — the app cannot read, list, or delete your emails. Tokens are stored locally with restricted permissions.
