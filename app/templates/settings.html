How it works
  • Installs a standard Linux/macOS crontab entry for the current user.
  • The cron job runs digest_cli.py (scrape + email) or scrape_cli.py (scrape only) from the project directory.
  • Output is logged to cron.log in the project root.
  • The web server does not need to be running for the cron job to work.
  • To inspect or edit manually: run crontab -e in your terminal.
